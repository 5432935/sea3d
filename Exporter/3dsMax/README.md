Installing
==

1. Open your 3dsMax 9 or higher
2. Click in MaxScript -> Run Script...
3. Select SEA3D.ms or Setup.ms to install
--

* SEA3D.ms **(Exporter)**
* SEA3DAnimation.ms **(Animation Editor)**
* SEA3DMaterial.ms **(Material Editor)**
* Setup.ms **(Installer)**
